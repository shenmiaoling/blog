通过 hexo serve 命令启动服务器

提交前的准备

cd public 进入 public 目录
hexo generate 编译代码后再将 public 文件进行提交
提交成功后可以访问 github 该项目里 Settings -> GitHub Pages -> Your site is published at https://userName.github.io/repositoriesName/
上面改地址即可访问线上的博客
